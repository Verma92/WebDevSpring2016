TaFree

TaFree is a website an interactive Event search website developed using web technologies like HTML5, CSS3, Bootstrap, Material Design Lite JQuery, Javascript, Angular.js, Nodejs. MongoDB is used as the database and REST architecture is used for interaction between client and server. Eventrbrite API was used for acquiring events data. JSON data was obtained from Eventbrite according to the query entered and GeoLocation of the user.
